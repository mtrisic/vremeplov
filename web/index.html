  • Type at the READY prompt — Esc = Break, Backspace = Delete,
    Tab = List — or paste a whole BASIC listing (Ctrl+V).
    • 
  
  • Tape: picking a .gtp or digitized .wav
    resets the machine, loads it, and RUNs it; Reload repeats that.
    • 
  
  • Reset gives a fresh machine at READY and forgets the tape.
    • 
  
  • Monitor: machine-language debugger — breakpoints, stepping,
    disassembly; type help at its prompt. Keys go to the
    machine unless the prompt has focus.
    • 
  
  • Record captures SAVEs and downloads them as .gtp
    or .wav audio.
    • 
  
  • State: Save downloads the whole machine as a snapshot file;
    Load resumes it exactly — even mid-game, even mid-tape-load.
    • 
  
  • Sound is the cassette-port speaker trick: the Galaksija had
    no sound chip, but games clicked and beeped through the tape output —
    plug in the virtual speaker and hear it.
